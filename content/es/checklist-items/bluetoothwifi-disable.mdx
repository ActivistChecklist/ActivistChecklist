Si vas a llevar tu teléfono, asumimos que lo vas a usar para comunicarte y que necesitas conexión celular, y que activar el modo avión probablemente no va a funcionar para ti. Por eso, lo mejor es **desactivar el Bluetooth y el Wi-Fi** para reducir el rastreo.

<Alert type="warning">
  **Advertencia para iPhone:** Deslizar hacia abajo y usar el "**centro de control**" para desactivar el Bluetooth/Wi-Fi no los apaga realmente. Solo te desconecta de las redes actuales, pero el teléfono sigue emitiendo señales que pueden ser rastreadas. Para apagarlos por completo, necesitas seguir las instrucciones que aparecen a continuación.
</Alert>

<HowTo title="Cómo desactivar el Bluetooth y el Wi-Fi">
  *En iPhone:*

  1. Configuración > Bluetooth > **Desactivar Bluetooth**
  2. Configuración > Wi-Fi > **Desactivar Wi-Fi**

  *En Android:*

  1. Configuración → Bluetooth → Avanzado (o menú de tres puntos) → **Desactivar Bluetooth** y **desactivar "Búsqueda de Bluetooth"**
  2. Configuración → Wi-Fi → Avanzado (o menú de tres puntos) → **Desactivar Wi-Fi** y **desactivar "Búsqueda" o "Búsqueda de Wi-Fi"**
</HowTo>
