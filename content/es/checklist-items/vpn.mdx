Cada vez que te conectas a internet (teléfono o computadora), tu proveedor de internet le revela tu ubicación aproximada a cada sitio o app que uses.

Sabemos que [las agencias policiales están usando herramientas para rastrear tu ubicación](https://www.404media.co/inside-ices-tool-to-monitor-phones-in-entire-neighborhoods/) usando datos recopilados de las apps que tienes y los anuncios que ves. Usar una VPN con funciones de bloqueo de anuncios activadas hace mucho más difícil que te rastreen. Además, los policías pueden conseguir una orden judicial para obtener tu tráfico de internet de tu proveedor.

Una VPN (Red Privada Virtual) ayuda a ocultar tu ubicación y hace un poco más difícil identificarte.

Una VPN con bloqueo de anuncios activado en tu celular puede ayudar a evitar que el gobierno rastree tu ubicación sin una orden judicial (a través de redes publicitarias).

<Alert type="warning">
  **Una VPN no te hace totalmente anónime.** Para navegación sensible que requiera mayor anonimato, considera [Tor Browser](https://activistchecklist.org/research) como un segundo navegador junto al que usas a diario.
</Alert>

**Opciones:** Todas estas son opciones muy confiables.

* [**Mullvad VPN**](https://mullvad.net/en/vpn) **(nuestra principal recomendación)**: Mejor privacidad porque no guarda la información de pago, pero tienes que pagar manualmente cada ciclo.
* [**IVPN**](https://www.ivpn.net/en/) puede ser más fácil porque se renueva automáticamente.
* [**Proton VPN**](https://protonvpn.com/) tiene un plan gratuito sólido, pero solo es para 1 dispositivo. Para obtener bloqueo de anuncios, necesitas un plan de pago. (Si usas Proton, mira nuestra [guía para usar Proton de forma segura](/proton/).)

**Desventajas de usar una VPN**

* Te vas a encontrar con más CAPTCHAs en los sitios web
* Algunos sitios web pueden bloquear el acceso por VPN, y vas a tener que desactivarla y acordarte de volver a activarla después
* Algunos servicios de streaming pueden no funcionar

Si notas un comportamiento extraño en un sitio web, intenta apagar la VPN temporalmente para ver si carga. Pero, como es fácil olvidarse de volver a encender la VPN, la mejor solución es activar [split tunneling](https://mullvad.net/en/help/split-tunneling-with-the-mullvad-app) en Mullvad. Puedes configurarla para que un navegador secundario se salte la VPN. Por ejemplo, si usas Brave como tu navegador principal, puedes configurar Chrome o Safari para que omitan la VPN y usar ese navegador solo cuando un sitio web no esté funcionando.

*Nota:* Debes usar una VPN confiable que no guarde registros de tu tráfico de internet y que se resista a las solicitudes del gobierno. Ya investigamos nuestras recomendaciones principales.
