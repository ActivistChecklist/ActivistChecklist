Las empresas tecnológicas (como Google) reciben con frecuencia [solicitudes gubernamentales](https://theintercept.com/2025/09/16/google-facebook-subpoena-ice-students-gaza/) para entregar [todos los datos de la cuenta](https://cldc.org/google-is-a-fcking-snitch/) de una persona activista. Esto suele incluir una "orden de silencio" que significa que la empresa no puede avisarle al usuario hasta meses o años después.

Las herramientas en la nube cifradas como [Proton Mail](https://proton.me/mail), [Drive](https://proton.me/drive/), [Docs](https://proton.me/drive/docs) y [Sheets](https://proton.me/drive/sheets) garantizan que la empresa no tenga copias legibles de tus datos para entregar. Si el gobierno los quiere, tendría que pedírtelos directamente a ti, lo que te daría aviso y la oportunidad de impugnarlo con apoyo legal.

<Alert type="warning">
  **Limitaciones del correo cifrado:** Proton cifra todo en sus servidores. Pero el cifrado solo protege ambos extremos si quien recibe el mensaje también usa correo cifrado. Si le escribes a una cuenta de Gmail, Google tiene una copia que puede entregar.

  **Para comunicaciones sensibles, usa siempre Signal** con los mensajes que desaparecen activados.
</Alert>

<HowTo title="Cómo configurar y usar Proton Docs y Proton Mail">
  **Crea una cuenta de Proton**

  <Alert type="info">
    **Sugerencia:** Elige un [nombre de usuario aleatorio](https://strongphrase.net/username) y un [nombre para mostrar aleatorio](https://strongphrase.net/identity). para que tengas la opción de no revelar que compartes esta dirección de correo.
  </Alert>

  1. **Regístrate** para obtener una [cuenta gratuita de Proton](https://account.proton.me/signup).
  2. **Verifica:** Cuando te pidan verificar que eres humano, elige la opción
  3. Cuando te pidan configurar tu número de teléfono / correo electrónico como método de recuperación, elige **Quizás más tarde.**

  **Usa Proton Drive/Docs/Sheets**

  * **Crea y edita** documentos de forma colaborativa, igual que con Google Docs. Alguien debe tener una cuenta de Proton para que se le pueda compartir directamente.
  * **Comparte de forma segura:** Usa "compartir por enlace" solo cuando sea necesario, establece una contraseña para el documento y configura el "enlace público" con una fecha de vencimiento.

  **Usa Proton Mail**

  * **Usa Proton Mail para:** cuentas de sitios web, boletines, comunicaciones públicas que requieran anonimato y trabajo de organización no sensible.
  * **No uses ningún correo electrónico para:** comunicaciones verdaderamente sensibles que puedan poner a alguien en riesgo legal.
  * **Qué se cifra y qué no:** Los mensajes entre usuarios de Proton Mail están automáticamente cifrados de extremo a extremo. Los mensajes a personas que usan otro proveedor de correo no estarán cifrados, pero puedes enviar un [correo protegido con contraseña](https://proton.me/support/password-protected-emails).
</HowTo>

[CryptPad](https://cryptpad.fr/) es otra opción popular de documentos cifrados, pero es muy difícil de usar. Si tiene las funciones que necesitas, logra el mismo resultado.
