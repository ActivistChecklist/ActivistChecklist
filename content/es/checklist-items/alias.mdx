En lugar de registrarte con tu nombre real, correo electrónico, teléfono, etc., puedes empezar a usar **identidades aleatorias y únicas para cada cuenta**.

Esto hace que sea más difícil para los atacantes (y las empresas tecnológicas) vincular tus cuentas entre sitios y crear un perfil a partir de todos esos datos.

<HowTo title="Cómo usar una identidad única para cada cuenta">
  Asegúrate de que cada uno de estos sea único para cada sitio, cuando sea posible:

  * **Nombre de visualización:** [Usa este generador de identidades aleatorias](https://strongphrase.net/identity).
  * **Nombre de usuario:** [Usa este generador de identidades](https://strongphrase.net/identity) o este [generador de nombres de usuario](https://strongphrase.net/username) específico.
  * **Foto de perfil:** Déjala en blanco o usa [este generador de identidades](https://strongphrase.net/identity). Si usas la misma foto en todos los sitios, un atacante puede hacer una "búsqueda inversa de imágenes" para encontrar todas tus cuentas.
  * **Dirección de correo electrónico** - Usa un servicio de alias de correo como:
    * [Qwacky para usar alias de correo @duck.com](https://github.com/Lanshuns/Qwacky) (gratis, extensión para Chrome/Firefox)
    * [Proton Mail](https://proton.me/support/pass-email-alias) (gratis/de pago)
    * [Addy.io](https://Addy.io) (gratis/de pago)
  * **Número de teléfono:** Intenta simplemente no ingresar un número de teléfono cuando te lo pidan. Si necesitas ingresar un número de teléfono solo para verificación al registrarte, intenta alquilar un número de [SMSPool](https://www.smspool.net/) ($0,50 por registro). Si necesitas un número de teléfono a largo plazo, considera [MySudo](https://anonyome.com/individuals/mysudo/).
</HowTo>
